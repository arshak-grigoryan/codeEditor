## Usage

codeEditor requires [node.js](https://nodejs.org/) to run.

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve

To serve in the browser - Runs webpack-dev-server

```sh
$ npm start
```

### Dev

Run application in development mode

```sh
$ npm run dev
```

### Watch

Watch application in development mode

```sh
$ npm run watch
```

### Build

Compile and build

```sh
$ npm run build
```
